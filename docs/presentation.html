Applied AI

Generating & Explaining CPNS Exam Questions

Problem → Architecture → Evaluation → Roadmap
A

The Opportunity

CPNS exam prep is a mass market — but quality practice questions and clear explanations are hand-authored, slow to produce, and never enough.
  • Scale the question bank. A small team seeds one question; the system drafts unlimited exam-style variants.
  • Tutor every learner. On-demand "why" for any question — explanations that didn't exist before.
  • Augment, not replace. Humans review; the model removes the blank-page cost of authoring.

C

Generation & Evaluation

Evaluation is built into the pipeline — every generation self-checks before it is accepted.

VERIFYSymbolic engine re-computes math → confirm / patch / correct
VALIDATEAnalogi spec checker + 3 retries reject malformed visuals pre-render
PARSEConstrained JSON decoding → zero unparseable outputs
D

What I Learned

  • Pair neural generation with a deterministic verifier — LLMs hallucinate facts like arithmetic
  • Match model class to task — encoder for retrieval, decoder for generation
  • Right-size infra — a small corpus didn't need a vector DB; flat cosine wins
  • Constrained decoding removes a whole class of parsing failures
E

Future Work
